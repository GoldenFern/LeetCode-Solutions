# LeetCode 73: 矩阵置零 (Set Matrix Zeroes)

## 1. 题目描述

给定一个 `m x n` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 **原地** 算法。

**示例 1:**

> 输入: matrix = [[1,1,1],[1,0,1],[1,1,1]]
> 输出: [[1,0,1],[0,0,0],[1,0,1]]

**示例 2:**

> 输入: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
> 输出: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]

**提示：**

* `m == matrix.length`
* `n == matrix[0].length`
* `1 <= m, n <= 200`
* $-2^{31} \le matrix[i][j] \le 2^{31} - 1$

---

## 2. 算法基本原理：原地标记法 (In-place Marking)

本题的核心挑战在于如何在不使用额外 $O(m+n)$ 空间的情况下，记录哪些行和列需要置零。

* **核心思想**：利用矩阵的**第一行**和**第一列**作为“备忘录”。
* **空间优化逻辑**：
  1. 先用两个布尔变量记录第一行和第一列本身是否包含 0。
  2. 遍历矩阵剩余部分（从 `matrix[1][1]` 开始），如果 `matrix[i][j] == 0`，则将对应的首行位置 `matrix[0][j]` 和首列位置 `matrix[i][0]` 设为 0。
  3. 根据首行和首列的标记，将内部元素置零。
  4. 最后根据第一步的布尔变量，单独处理第一行和第一列。
* **复杂度分析**：
  * **时间复杂度**：$O(m \times n)$，需要遍历矩阵两次。
  * **空间复杂度**：$O(1)$，仅使用了常数个额外变量。
